vim-corporati
=============

A simple vim colorscheme based on Corporation, modified over at
http://bytefluent.com/vivify/

![Corporati in gVim"](corporati.png)

How to install
--------------

    $ cp colors/corporati.vim ~/.vim/colors/

then in vim

    :colors corporati
